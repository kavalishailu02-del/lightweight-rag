# Local PDF Q&A using PyMuPDF4LLM and Llama.cpp

## Overview

This project allows users to upload PDF files and ask questions about their content using a local Large Language Model.

## Features

- PDF extraction using PyMuPDF4LLM
- BM25 retrieval
- Local LLM using llama.cpp
- Streamlit web interface
- Offline document question answering

## Tech Stack

- Python
- Streamlit
- PyMuPDF4LLM
- llama.cpp
- BM25

## Run Project

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
